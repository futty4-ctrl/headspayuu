# 画像ファイル配置ガイド

## 配置済み画像（カルーセル・当店について）

| ファイル名 | 割り当て先 |
|-----------|-----------|
| `about-portrait.png` | 当店についてセクション（プロフィール） |
| `card-2-newcomer.png` | 御新規様特典 |
| `card-3-voices.png` | Customer Voices（お客様の声） |
| `card-4-giftcard.png` | ギフトカード |
| `card-5-osaka.png` | DRY HEADSPA ONLY（大阪） |
| `card-7-booking.png` | ネット予約システム |
| `menu-scalp.png` | ヘッドスパ×育毛 / 施術メニュー |
| `menu-color.png` | ONLINE STORE / ヘアカラー施術 |

## 追加で配置するとよい画像

| ファイル名 | 割り当て先 |
|-----------|-----------|
| `hero.jpg` | ヒーローセクション背景 |
| `essence-01.jpg` ～ `essence-10.jpg` | 10のこだわり（各アイテム） |
| `before.jpg` | ビフォーアフター（施術前） |
| `after.jpg` | ビフォーアフター（施術後） |

画像が無い箇所はプレースホルダー（グレー背景）が表示されます。
